## 專案背景

KM3C 專注於提供極致操控體驗的電競鍵盤與滑鼠，結合專業級性能與簡約設計，為玩家與創作者帶來流暢、穩定的操作手感。
除了核心的鍵鼠產品外，我們也提供多款高品質耳機、滑鼠墊與其他周邊配件， 滿足從遊戲戰場到日常工作的各種需求。 KM3C 以「精準、舒適、耐用」為理念，讓每一次點擊與操作，都成為勝利的起點。

## LOGO命名與設計
KM3C 以鍵盤跟滑鼠的英文單字的第一個字母為題，也是為了強調本網站所賣的主要商品，能夠讓使用者了解我們能提供的服務，並帶來美好的商品體驗。

## 未來計畫

規劃將以 Vue3 框架進行重構:
- 元件化設計: 將會重複利用到的組件，如 Header, Footer 以及 商品卡 等等的 UI 拆成單獨組件，提升可複用性
- 狀態管理: 利用 pinia 將會員以及其他資料進行集中管理
- 前後端串接: 使用 PHP 與 MySql 來形成網站的資料流
